06 Tokens Studio — Import Instructions
1
Open Figma and install Tokens Studio for Figma from the Community plugins if not already installed.
2
Open the Tokens Studio plugin. In the top-right menu, select Load from file / folder or click the ... overflow menu → Import.
3
Select the file fm-synth-tokens.json provided alongside this reference. The plugin will parse it into three token sets: global, light, and dark.
4
Enable the token sets: check global as always-on. Switch between light and dark to change themes. The two themes are pre-configured in the JSON.
5
To apply a token: select any element in Figma, then in Tokens Studio click the token value. It will bind to the fill, stroke, or text color of the selected layer. Semantic tokens (e.g. bg/panel) are aliased to primitives — editing a primitive updates all aliases automatically.
6
Optional: use Sync → Figma Variables to push tokens into native Figma Variables. This enables the Figma-native variable inspector and mode switching directly in the design panel.
